# 冒险者公会制度

冒险者公会是阳界范围内的中立职业组织。它不隶属于地方政权、贵族、骑士团或家族势力，但会与城门事务厅、城防、骑队、货栈、商队和疗愈院长期合作。

## 公会职能

公会负责把各类报事转化为可执行任务，并管理成员资质。

主要职责包括：

- 发布、撤回、分派任务；
- 管理会员牌级；
- 接收、审核、归档回执；
- 判定任务完成情况；
- 判断任务是否升级；
- 决定是否转交本地骑队、城防或更高层公会；
- 维持冒险者与委托人之间的基本秩序。

主角团作为公会成员，正式委托应从冒险者公会接取。城门事务厅、货栈、药铺、商队或牧户可以是任务来源和执行地点，但不是正式接牌处。

## 城门事务厅

城门事务厅属于本地行政系统，不属于冒险者公会。

它负责：

- 通行登记；
- 商队、货车、牲口、旅人进出记录；
- 路税、货损、报案、失物、纠纷初记；
- 城门外短距离聚居点、牧点、旧棚、旧驿道支线的报备；
- 把需要外勤核查的事项转给冒险者公会。

标准流程：

```text
报事者向事务厅报事
→ 事务厅初记
→ 材料转给冒险者公会
→ 公会判断任务等级并挂牌或分派
→ 冒险者执行任务
→ 冒险者回公会交回执
→ 公会决定回传、并入更高任务或转交骑队
```

## 四门事务方向

| 门区 | 常见事务 |
|---|---|
| 北门事务厅 | 北门路税、药材短护送、旧驿道路段补记、牧点旧棚报事、北向短程护送 |
| 东门事务厅 | 东关货栈、东北线商队、车队入城、货损、商路报事 |
| 南门事务厅 | 南下商队、长途驿路、粮货车队、南向驿站 |
| 西门事务厅 | 手工业区、工坊、仓储、旧仓、坡地支路、城郊短路 |

## 任务等级

| 等级 | 特征 | 例子 |
|---|---|---|
| 低级任务 | 风险可控，不要求正面处理明确危险 | 找人、搬货、认路、送信、普通护送、文书跑腿、简单问讯 |
| 中级任务 | 已有明确风险，需要成熟成员、多人协作或外勤书记见证 | 危险生物、武装冲突、遗迹异常、夜间驻守、异常兽扰 |
| 高级任务 | 风险高、牵涉权力层级或古代遗迹核心 | 魔兽、骑士团、贵族领地、高危险遗迹、跨区域商道安全 |
| 骑队任务 | 影响道路安全、边境秩序或需要武装力量介入 | 封道、武装巡查、清剿、商道安全处置 |

骑队任务不是普通会员等级，而是公会与本地骑队的协作机制。

## 预核牌与灰线牌

某些任务表面仍是低级外围核查，实则可能牵涉异常。这类任务可称为预核牌、灰线牌或限时低级牌。

它们通常要求成员做：

- 问讯；
- 补路图；
- 核对目击位置；
- 标注活动范围；
- 记录异常痕迹；
- 把普通人和危险点隔开；
- 带回可用回执；
- 交由中级组或更高层接手。

这类任务不是让低级成员冒险处置危险。完成得稳，可以记入升牌考核。

## 公会人员

| 人员 | 职责 |
|---|---|
| 柜台书记员 | 挂牌、销牌、结钱、接收普通回执、转交异常材料 |
| 灰袍管理员 | 处理预核牌和灰线任务，审查异常回执，提出升牌建议 |
| 外勤书记 | 到现场记录、见证、安抚报事者，阻止普通人靠近危险区域 |
| 顾问会成员 | 复核异常回执，判断任务升级，决定是否给升牌或转交骑队 |
| 正式会员 | 按低级、中级、高级牌级接取对应任务 |

## 中级试牌

中级试牌是低级与正式中级之间的过渡牌。

它允许成员参与中级任务的外围和预核环节，但不能独立处置中级风险。

| 可做 | 不可做 |
|---|---|
| 参与中级异常核查 | 单独清剿 |
| 做外围预核 | 单独守夜 |
| 问讯、补路图、路段标记 | 独立处理遗迹异常 |
| 协助中级组 | 独立接高级任务 |
| 在外勤书记或中级成员在场时参与灰线任务 | 接骑队任务或进入遗迹核心 |

试牌期可设为三个月。越界则收回，回执稳定才可转为正式中级牌。

## 升牌机制

公会牌不是按任务数量提升，而是按综合记录换牌。

低级升中级通常需要：

1. 稳定的销牌记录；
2. 至少一次中级相关实绩；
3. 一个可担责的人签名。

公会看重的是成员能否按时回牌、少出事故、不擅自扩大任务范围，并把自己看不懂的东西如实写成看不懂。

## 灰蜡点与灰蜡薄夹

灰蜡点是公会内部文书标记，表示这份回执不能直接进普通外账，需要转入内部复核。

灰蜡薄夹常用于：

- 非公开异常预警回执；
- 灰线任务；
- 旧路、旧棚、地面异常、遗迹外围材料；
- 需要中级组、高级成员或顾问会复核的回执；
- 可能影响成员升牌考核的任务记录。

## 本地骑队与公会

本地骑队属于风岭城本地秩序体系，不是公会内部部门，也不是某个家族私兵。

公会负责收集线索、判断风险和派遣外围协助。骑队负责道路封控、武装巡查、商道安全和紧急压场。

瓦尔凯恩家族可以凭军务经验和本地声望提供有限顾问协助，但不控制公会，不决定任务权，也不决定会员升牌。
