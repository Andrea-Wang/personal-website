# 界膜、裂隙与稳定场

界膜是阳界与阴界之间的相位分隔结构。它隔开两种相位的物质与能量，使两个世界能够在镜像位置上并存，而不发生直接重叠或湮灭。

## 信息卡

| 项目 | 内容 |
|---|---|
| 名称 | 界膜 / Veil |
| 位置 | 阳界与阴界之间，靠近核心层 |
| 功能 | 相位分离、能量转换、世界稳定 |
| 相关装置 | 古代界膜稳定系统 |
| 能源来源 | 封装原太阳、轨道阵列、古代节点、星球核心 |
| 失稳表现 | 光照异常、遗迹反应、变异显形、裂隙、空间重叠 |

## 相位分离

阳界、界膜与阴界可简化为三种相位。界膜维持相位分离。只要分离有效，两界即使在镜像位置上重合，也不会互相湮灭。

## 界膜稳定系统

界膜不是天然永恒的墙。它由第三纪元末期的相位稳定装置维持，并依赖外层稳定场持续覆盖。

稳定场的主要能源来自被封装的原太阳。恒星能源经由中继卫星、轨道阵列及同类外层装置吸收、转化、定向辐照，再分别投向阳界与阴界。

因此，稳定场的方向不是从核心向外扩散，而是从各自世界的外侧太空向界中心扩散。两界在机制上对称。

## 界膜稳定度

界膜稳定度指某一区域、某一时期内稳定场维持有效相位覆盖的程度。

| 稳定度 | 状态 |
|---|---|
| 高 | 稳定场覆盖完整，从高处到低处都受到保护 |
| 下降 | 覆盖向高处、外侧、古代节点和共振区域退缩 |
| 极低 | 大范围覆盖余量耗尽，只剩局部节点维持最低稳定 |
| 崩坏 | 裂隙出现，局部空间发生相位破损 |

稳定度不是能量百分比，也不按高度线性递减。它更像波场，受到地形、节点、材料、遗迹、核心状态和文明活动共同影响。

## 非线性覆盖

同样高度的地区可能呈现完全不同的稳定状态。

影响因素包括：

- 外层稳定辐照强度；
- 地形高度；
- 是否靠近古代节点；
- 当地材料是否具有相位传导性；
- 遗迹结构是否仍能共振；
- 星球核心能量状态；
- 附近是否存在裂隙或能量风暴；
- 文明是否大量调用相位能量。

因此，高山、旧节点、古代遗迹有时比低地更稳定；低地城市若建在旧节点上，也可能长期安全。

## 稳定度下降的阶段

| 阶段 | 表现 |
|---|---|
| 轻度下降 | 天气、光照、动物行为、遗迹反应出现细微异常 |
| 中度下降 | 变异生命隐性特征开始显现 |
| 重度下降 | 局部裂隙出现，周边生命和能量流受到扰动 |
| 极端下降 | 空间短暂重叠，出现能量风暴、时间错位、跨界误入、遗迹复醒 |

## 裂隙类型

裂隙是界膜局部破损后形成的相位通道或空间伤口。

| 类型 | 描述 |
|---|---|
| 微型裂隙 | 主要影响能量流与局部感知 |
| 中型裂隙 | 可能允许生物误入或短暂通过 |
| 巨型裂隙 | 两界短时重叠，伴随强烈能量风暴 |

裂隙并不总是稳定的门。多数裂隙更像短暂撕开的伤口，危险来自相位波动、环境错位和周边生命失控。
