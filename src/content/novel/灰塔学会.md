# 灰塔学会

## 基本定位

灰塔学会是黑松北境王国本地最重要的遗迹研究、星铁研究与界膜节点维护机构。

它不是星冠学会的地方分支，也不是王国事务署的下属机关。它扎根于北境旧路、灰脊山、岩关、乌鸦王陵和界膜稳定装置，是一套长期维护、复核和封存危险知识的专业体系。

在黑松北境王国境内，凡涉及星铁、北境遗迹、乌鸦王留下的装置、旧路节点、岩关、灰脊山王陵和界膜稳定装置，灰塔学会拥有高于星冠学会的实际权限和知识优势。

> 星冠学会知道许多旧文献。  
> 灰塔学会知道这些旧文献对应的装置、节点和风险。

## 组织气质

灰塔不以政治斗争为目标，也不主动经营宫廷、贵族和学会派系。

它的气质是冷静、克制、技术化、少言、重复核。灰塔成员更像长期守在仪器、阵列、旧路和封卷旁的人：记录数值，核对痕迹，复查材料，校正波动。

## 历史来源

灰塔学会源头可追溯至乌鸦王时代。

灰脊山、岩关、灰塔与乌鸦王陵不是孤立地点，而是北境守望链条的一部分。灰塔早期的功能接近维护者和技术复核者，后来逐渐制度化，成为黑松北境王国最了解北境遗迹真相的机构。

## 核心研究领域

### 星铁研究

灰塔掌握星铁的来源、性质、传导特征、装置兼容性、拒绝反应和相位响应机制。

星铁不是普通稀有金属。它与界膜稳定装置、乌鸦王遗物和旧系统识别机制有关。乌鸦王生前所佩的王羽扣便由星铁铸成，表面上是王者旧物，实际可被界膜装置识别。

### 北境遗迹研究

对普通人而言，灰脊山旧路、岩关、乌鸦王陵、旧军道、废弃矿洞和古代节点只是散落各处的遗址。

对灰塔而言，它们属于同一个更大的系统。

灰塔知道哪些遗迹只是表层旧址，哪些位置仍残留有效结构，哪些地方不能开挖，哪些石构不能移动，哪些封段并不是因为塌方或山雪。

### 乌鸦王装置研究

乌鸦王陵不是普通墓穴，而是以王陵外壳掩盖界膜稳定装置的特殊遗址。

陵中大量遗物兼具纪念、识别、封存和继承资格验证等功能。灰塔对乌鸦王装置的了解来自技术传承和长期维护积累。

### 界膜稳定装置研究

灰塔作为机构，保存着界膜稳定装置的完整理论、运行逻辑、维护流程、风险判断和历史记录。

灰塔知道：

- 界膜为何存在；
- 稳定装置如何维持相位分离；
- 节点失稳会造成什么后果；
- 星铁在装置体系中的作用；
- 乌鸦王陵与灰脊山节点链条之间的关系；
- 哪些异常属于裂隙前兆或装置残响；
- 什么信息公开后会引发误用、争夺或灾难。

## 权限体系

在黑松北境王国境内，灰塔的权限具有领域优先性。

| 身份 | 凭证 | 主要权限 |
|---|---|---|
| 王国事务署管护吏 | 署印 | 行政管护、登记、调度、出入核准 |
| 星冠学会正员 | 星印 | 文献谱系、旧帝国学统、藏卷校勘 |
| 灰塔学会认证者 | 塔印 | 北境旧址、遗迹结构、星铁、界膜节点、异常复核 |
| 外聘核录者 | 灰牌 | 普通阅览区、公开副卷、限定类目核录 |

星冠学会在文献和学统上仍重要；灰塔则在北境遗迹核心、星铁反应、界膜稳定装置和裂隙风险上掌握最后一道技术门槛。

## 与星冠学会的关系

灰塔与星冠不是简单敌对。

| 项目 | 星冠学会 | 灰塔学会 |
|---|---|---|
| 核心领域 | 文献、学统、目录、校勘 | 星铁、遗迹、节点、装置维护 |
| 权威来源 | 旧帝国学统、星冠旧藏、赐姓家族 | 灰塔旧档、节点实测、星铁研究 |
| 大陆影响 | 更广 | 更窄 |
| 北境遗迹事务 | 低于灰塔 | 高于星冠 |
| 风险 | 知识门阀化 | 技术壁垒化 |

星冠学会知道一份古卷的版本、抄写谱系和学术归类；灰塔学会知道卷中地名对应哪个节点，那个节点是否仍有反应，以及贸然进入会引发什么后果。

## 与王国事务署的关系

王国事务署提供制度外壳，灰塔提供技术裁定。

> 王国事务署决定这件事该如何登记。  
> 灰塔判断这件事到底是什么。  
> 星冠说明它在文献谱系里属于哪一类。

## 与公会、边军的关系

公会负责外围任务、调查委托、异常回执和现场材料初筛。边军负责封锁、巡查、护送、压场和清剿。灰塔负责判断现象背后的真正性质。

低级公会成员通常不会直接接触“界膜”“星铁”“旧系统”这些核心词，只会接到看似普通的查明情况、护送、封路、找人、清理异常等任务。

## 成员类型

| 类型 | 职责 |
|---|---|
| 阵列维护师 | 界膜节点、旧阵列、外环结构和装置残余校正 |
| 星铁研究者 | 星铁材料、构件、残响、拒绝反应和识别机制 |
| 旧址复核师 | 判断旧路、废矿、残墙是否属于远古节点体系 |
| 封卷管理员 | 管理灰塔旧档、封存卷、抹名记录和封段档案 |
| 节点巡检员 | 巡查灰脊山、岩关、旧军道和疑似裂隙区域 |
| 外环阵师 | 重大界膜异常时负责外环校正和稳定场控制 |
| 灰塔认证者 | 持塔印进入灰塔相关类目和深档准入系统 |

## 社交边界

灰塔不是孤塔。成员可以有朋友、同伴、爱人、师生、同僚和长期合作对象。

但灰塔有一条清楚的边界：

> 关系可以靠近，权限不能靠近。

私人信任不会自动等于资料准入。共同任务不会自动等于核心解释权。感情关系也不会让灰塔成员随口泄露研究机密。

## 核心矛盾

灰塔学会的核心矛盾是：

> 它知道最多，也最有理由沉默。

它守住过危险，也压下过真相。它维护过界膜，也维护过少数人掌握真相的旧秩序。

灰塔不是反派。它的沉默有现实理由：星铁会被争夺，装置会被误用，界膜知识会引来黑羽议会一类势力，旧路节点若被错误开启，可能造成裂隙、能量风暴、时间错位甚至跨界灾难。

问题在于：当一个机构长期以“危险”为理由替所有人决定什么能知道、什么不能知道时，它本身也会成为壁垒。
